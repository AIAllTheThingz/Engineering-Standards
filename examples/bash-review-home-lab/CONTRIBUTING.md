# Contributing

Keep this example deterministic, synthetic, findings-only, and secret-free.
Never source or execute the unsafe Bash sample. Use `example.invalid` and
synthetic identifiers, update fixtures and illustrative contracts together,
and run:

```powershell
pwsh -NoProfile -File examples/bash-review-home-lab/tools/Test-Demo.ps1
```

Report unavailable checks as `NotRun`.
