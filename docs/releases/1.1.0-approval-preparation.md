## Final Approval Effectiveness

The release approval, `GOV-2026-001` disposition, tag authorization, and publication authorization become effective only when the approval-finalization pull request:

1. has no commits after the reviewed final head,
2. passes `Governance / Governance validation`,
3. has formal GitHub `APPROVED` reviews from both `megad00die` and `mezuccolini`,
4. has no unresolved review threads, and
5. is merged into protected `master`.

The authoritative final approval evidence is the GitHub pull-request review history and merge event for that final head. Exact final review IDs and GitHub `submitted_at` timestamps remain in GitHub's durable review history and do not require a post-approval commit to this file.
